# YouTube In-Player Controller

YouTubeの再生画面内に直接、シーク（時間調整）や再生速度の微調整を行えるパネルを追加するChrome拡張機能です。
最新Ver 1.1

## 特徴
- 0.1秒単位のスムーズシーク
- 0.01倍単位の再生速度調整
- 自動ループ再生機能
- YouTubeのプレイヤーに直接UIを統合

## インストール方法
Chromeウェブストアを経由しないため、以下の手順で手動インストールをお願いします。

1. このページの右上にある緑色の「Code」ボタンをクリックし、「Download ZIP」を選択して保存・解凍します。右側のRelease欄からもDLできます
2. 拡張機能の管理画面を開きます。(ChromeのURLバーに `chrome://extensions/` と入力するか、右上のメニューから「拡張機能→拡張機能を管理」を開く)
3. 画面右上の **「デベロッパーモード」** のスイッチをオンにします。
4. 左上に現れる **「パッケージ化されていない拡張機能を読み込む」** をクリックし、解凍したフォルダ（`manifest.json` が入っているフォルダ）を選択します。
5. YouTubeを開き、動画にマウスを乗せると下部にコントローラーが表示されます。
